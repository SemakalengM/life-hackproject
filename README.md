# life-hackproject
1. Welcome Screen

When the app launches, users are presented with a welcome screen that:

Displays the app title and a short description
Introduces the purpose of the quiz (identifying real hacks vs myths)
Provides a “Start Quiz” button

2. Flashcard Question Screen

This is the core functionality of the app.

The app displays one life hack statement at a time
Users are given two answer options:
Hack (True) → if they believe the statement is correct
Myth (False) → if they believe it is incorrect
Interaction Flow:
The user reads the statement
Selects either Hack or Myth
The app immediately:
Checks the answer
Displays feedback such as:
“Correct!”
“Wrong! ”
The app updates the user’s score internally

This process continues in a loop until all questions are answered.

3. Score Screen

After completing all flashcards:

The app calculates the total score
Displays:
Number of correct answers
Total number of questions
Provides personalised feedback based on performance:
High score: “Great job!” or “Master Hacker!”
Average score: “Good effort!”
Low score: “Keep practising!”

4. Review Screen

This screen allows users to reinforce learning by reviewing all content.

Displays:
Each question
The correct answer (Hack/Myth)
A short explanation


How Users Interact with the App

Users interact with the app through a simple and intuitive interface:

Tap buttons to select answers (Hack/Myth)
Navigate screens using Start, Next, and Review buttons
Read feedback instantly after each answer
Track progress through final score results
Review content to improve understanding

The app requires no login or prior knowledge, making it accessible to all users
